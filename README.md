## Instalation

To install local-ci go to your repository and run:
    
    pip install local-ci
    local-ci install

Create file `.local-ci.yml` configuration file (example: https://github.com/8tm/local-ci/.local-ci.yml)  

## local-ci usage

### run local-ci manually

    local-ci

### Install local-ci

    local-ci install

### Uninstall local-ci

    local-ci uninstall
